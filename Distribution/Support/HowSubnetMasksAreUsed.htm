How Subnet Masks Are Used

 

    Subnet masks are used to determine the Network ID. Routers do this by performing a binary 'AND' function on the given IP Address and the subnet mask (routers need to do this to figure out which network to send a given packet to). Remember that every IP Address is actually a binary number (a string of 32 ones and zeros). The same is true of a subnet mask.

1100 0000.1010 1000.0000 1010.0010 0101  - IP Address (192.168.10.37)
1111 1111.1111 1111.1111 1111.0000 0000  - Subnet Mask (255.255.255.0)
------------------------------------------------
1100 0000.1010 1000.0000 1010.0000 0000  - Network ID (192.168.10.0)
====== Network Part ======  | Host Part

Notice that each bit of the Network ID is only a one if the corresponding bits of the subnet mask and the IP Address are both ones.

Non-Routable IP Address

Some IP Addresses are ignored by routers; the routers will not forward packets to or from such IP Addresses. These type of IP Addresses are called 'non-routable' IP Addresses. Below is the list of non-routable IP Addresses categorized by IP Address Class.

Class A - 10.0.0.0 - 10.255.255.255
Class B - 172.16.0.0 - 172.31.255.255
Class C - 192.168.0.0 - 192.168.255.255
